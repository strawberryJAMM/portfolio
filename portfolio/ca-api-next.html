Project Name

CA API Portal "Next"

UI Concept Design    Wireframes   

UI Concept Design for CA Technologies' next generation Cloud Hosted API Security Portal Service.

Project Details
Mar 2014

During this one month contract project a low-fidelity, wire-frame based, interactive design prototype was created using Balsamiq Mockups in "Sketch" mode for CA's new "cloud computing" API Portal service. The design was intended to serve as an internal promotional supplement to the currently ongoing, but largely back-end, development work by offering a demonstration of one possible "end-game vision" for the product.

A key requirement of the new API Portal service was support for access by mobile devices. To feature this requirement, the prototype included "responsive web" variations of major application views for both smartphone and tablet form factors.
